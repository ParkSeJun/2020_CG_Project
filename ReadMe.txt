Program Files(x86)으로 설치 부탁
Lib, Include 파일 연동법
1. 프로젝트/속성
2. VC++ 디렉터리
3. 포함 디렉터리 제일 왼쪽 삼각형 클릭후 편집버튼 클릭
4. 포함 디렉터리 창이 뜨면 줄추가 버튼 클릭
5. ... 모양클릭
6. Microsoft DirectX SDK (June 2010)->Include 폴더 선택

7. 라이브러리 디렉터리 제일 왼쪽 삼각형 클릭후 편집 버튼 클릭
8. 라이브러리 디렉터리 창이 뜨면 줄추가 버튼 클릭
9. ... 모양클릭 
10. Microsoft DirectX SDK (June 2010)-> lib -> x86 폴더 선택

11. 적용 및 확인 버튼 클릭

- 조작법 - 
wasd: 방향키
1, 자유시점
2. 쿼터뷰 시점(플레이어 기준)으로 변환
마우스움직임으로 카메라 회전 적용
마우스 좌우 클릭: 기준이 될 객체의 회전
Space: 기준이 될 객체 전진

카메라가 로봇의 일정 거리내에 있을 경우 카메라 방향으로 회전
로봇이 카메라를 바라본 뒤, 일정 거리에 있을 경우 로봇이 카메라를 쫒아옴

쿼터뷰 시점을 사용하면 카메라는 객체의 뒤에 위치됨
이 경우 로봇이 카메라를 따라 오는게 아니라 생성된 객체를 기준으로 위의 
조건들이 적용됨

바닥에 장애물이 있는게 아니라 바닥을 장애물로 바꿔(하이트맵 적용)
모든 객체들이 하이트맵을 타게 됨
